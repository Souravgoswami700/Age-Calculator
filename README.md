
Age Calculator is a modern, responsive web application that accurately calculates your exact age in years, months, and days from your date of birth. Features beautiful glassmorphism design with smooth animations and precise date calculations.

✨ Features
✅ Precise age calculation (handles month/day borrowing correctly)

✅ Real-time auto-calculation on date input

✅ Future date validation (prevents invalid DOBs)

✅ Plural handling (1 year vs years, etc.)

✅ Glassmorphism UI with gradient backgrounds

✅ Mobile responsive design

✅ Smooth hover animations and focus states

✅ Separated HTML/CSS/JS files for clean architecture

🎯 How to Use
.Select your date of birth using the date picker

.Age calculates automatically (or click "Calculate My Age")

.View exact age in years, months, and days

.Try edge cases like leap years or birthdays today!
